# GitHub リポジトリ作成手順

## 手順

1. https://github.com/new を開く
2. 以下を入力:
   - **Repository name**: `kinyu-shinchaku` （または任意の名前）
   - **Visibility**: Private（会社業務なので非公開推奨）
   - ⚠️ 「Initialize this repository with a README」は **チェックしない**
3. 「Create repository」をクリック
4. 作成後に表示される URL（例: `https://github.com/yourname/kinyu-shinchaku.git`）をコピーしてClaude に共有
